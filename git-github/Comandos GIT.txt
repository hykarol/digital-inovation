COMANDOS BÁSICOS GIT
- git init - (Cria um novo repositório, inicializa o repositório)
- git add * - Processo para realizar o commit
- git commit -m "string passada para ter ideia do que é o commit"
- git status - informa o estado dos arquivos
- git config --global user.propriedade a ser inserida por ex: email, usuario.
- git config --global --unset user.propriedade a ser retirada.
- git remote add origin + URL - Adiciona um repositorio ao GitHub
- git push origin master - Também adiciona ou atualiza o repositorio.
- git remote -v - lista os repositorios existentes no GitHub
- git pull origin master
- git pull origin master - puxa o conteúdo atual no repositorio, para corrigir conflitos de merge.
- git clone + link_github a ser baixado.

Tracked e Untracked

Tracked - Arquivos que o GIT tem conhecimento.
Unmodified - Arquivo ainda não modificado.
Modified - Já foi modificado.
Staged - Arquivos em preparação para ir para outro agrupamento.

Untracked - Arquivos que o GIT não tem conhecimento.

Repositorios - Working Directory -> Staging Ares -> Local Repository
